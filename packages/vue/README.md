# @vjs-ui/vue

> VJS-UI Vue 3 适配器和组件库

## 功能

- 372个企业级组件
- Vue 3深度集成
- TypeScript类型支持
- Tree Shaking
- 自动按需导入

## 安装

```bash
pnpm add @vjs-ui/vue
```

## 使用

```vue
<template>
  <VButton type="primary">Click Me</VButton>
</template>

<script setup>
import { VButton } from '@vjs-ui/vue'
</script>
```
